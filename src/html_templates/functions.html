Available functions

Ping status service

To be able to use the ping status service, the desired hosts must first be added in settings. When all hosts have been added to settings, press the Restart ping service button below. This will update the ping service with the new hosts and report the ping status back to the Zipato cloud service.

Power on API command

The poweron command will attempt to start a host by sending it a wake on LAN packet.

Requires This command requires wakeonlan to be installed on the server running Zipato extension server .

Power off API command

To be able to use the poweroff command, the host must first be added in settings. The poweroff command will attempt to shut down a host by logging in to it and running the shutdown -h now.

Requires For this to work, a pair of authentication keys will have to be set up between the Zipato extension server and the target host.
